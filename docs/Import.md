# Import.md — outdated

This feature request has been implemented:

- **`make import ARGS="<slug>"`** — fills missing species in a family using the local Ollama model
- **`make fetch ARGS="<slug>"`** — fetches real species data from GBIF, enriched with Wikipedia
- **`make enrich`** — batch Wikipedia enrichment for all species

See `Coverage.md` for the current import tools documentation.

## Known Issues

### Vite OOM / large JSON
- `unified-taxonomy.json` is ~500MB+ now with 481k nodes
- Changed to `fetch()` at runtime instead of `import` — this avoids the Vite transform buffer crash
- `NODE_OPTIONS="--max-old-space-size=8192"` needed for both `npm run dev` and `npm run build`
- Dev script in `package.json` updated: `"dev": "NODE_OPTIONS=... sh scripts/buildData.sh && NODE_OPTIONS=... vite"`

### Remaining phyla not in GBIF
- Gastrotricha, Phoronida, Priapulida, Loricifera, Gnathostomulida, Entoprocta, Onychophora, Xenacoelomorpha — returned 0 classes from GBIF scout
- Would need manual taxonomy treatment (not handled by importClass.ts pipeline)

### reportPhyla.ts bug
- Shows Mollusca with only 1 class/8 families (not picking up new classes)
